# AI-Assignment-2

## Overview
This repository contains implementations of Artificial Intelligence concepts including agent design, CAPTCHA verification, Turing Test simulation, and Uninformed Search Algorithms.

The assignment demonstrates how different AI techniques and search strategies can be implemented using Python.

---

## Contents

### 1. Search Algorithms
The following uninformed search algorithms are implemented:

- **BFS (Breadth First Search)** – Traverses nodes level by level using a queue.
- **DFS (Depth First Search)** – Traverses nodes deeply using recursion.
- **Depth Limited DFS** – DFS with a predefined depth limit.
- **Iterative Deepening DFS** – Combines DFS and BFS advantages by gradually increasing depth.

Files:
- `BFS.py`
- `DFS.py`
- `DepthLimited_DFS.py`
- `IterativeDeepening_DFS.py`
